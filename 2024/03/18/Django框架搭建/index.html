一、Anaconda 虚拟环境安装

1. 安装url

Free Download | Anaconda

2. 环境配置 path

一般安装完成之后自动配置，自己看一眼，基本不需要配置。

3. 查看已有环境

conda env list

4. 新建环境

conda create -n mypython37 python=3.7

mypython37为环境名字，python=3.7版本号

5. 使用激活or切换

conda activate mypython37
